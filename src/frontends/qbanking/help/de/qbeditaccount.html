Kontoeinstellungen ändern

Dieser Dialog ermöglicht die Konfiguration eines Kontos.

Dieser Dialog enthält verschiedene Seiten:

  • Einleitung (enthält allgemeine Hinweise zum Dialog) [immer vorhanden]
  • Grundeinstellungen (allgemeine Einstellungen zum Konto) [immer vorhanden]
  • eine optionale Anwendungs-Seite (z.B. QBankManager, hier können Sie - falls diese Seite vorhanden ist - Anwendungs-spezifische Einstellungen vornehmen)
  • eine optionale Seite des Banking-Moduls (z.B. HBCI, hier können dann beispielsweise HBCI-spezifische Einstellungen vorgenommen werden)

Generelle Konto-Einstellungen

Hier können Sie generelle Einstellungen an Ihrem Konto vornehmen.

Bank-Einstellungen

Hier werden Einstellungen bezüglich der Bank vorgenommen.
Feld Erläuterung
Staat Wählen Sie den Staat, in dem die Bank ihren Sitz hat.
Bankleitzahl Geben Sie hier für deutsche Banken die Bankleitzahl ein. Für Staaten, in denen keine Bankleitzahlen üblich sind (bzw. falls Sie sie nicht kennen), geben Sie hier 00000000 ein.
Bankname Hier können Sie den Namen der Bank eingeben. Dieses Feld dient Ihnen später beispielweise bei Berichten zur besseren Identifizierung der Bank.

Verfügbare Benutzer / ausgewählte Benutzer

Manche Banking-Module arbeiten Benutzer-orientiert (wie HBCI, OFX etc). Diese Module müssen wissen, unter welchem Benutzer Sie sich bei der Bank anmelden wollen, wenn Sie Aufträge ausführen wollen.

Auf der linken Seite (verfügbare Benutzer) finden Sie die Benutzer, die Sie dem Konto zuordnen können.

Auf der rechten Seite finden Sie die Benutzer, die bereits dem Konto zugeordnet wurden (entweder durch Sie oder automatisch durch das Banking-Modul).
